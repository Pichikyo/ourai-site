VRChat｜産業XR

京セラ｜B2B展示から地域連携まで、
5つのワールドで続く産業XRパートナーシップ

産業XRバーチャル展示B2Bショールームイベント開催プランニング
期間：2022〜継続中

JIMTOF2022出展に向けた工作機器B2Bショールーム「Kyocera Tool World」を皮切りに、レーザーコンセプト製品展示（ジャパンドローン2023連動）、高耐久スマートフォン展示・物流体験「Kyocera Mobile World」、製造工程を体験できる「ファインセラミックスワールド」、京都府京田辺市・甘南備山モチーフ「ワールドポータル by Kyocera」まで、5つのワールドを継続的に制作しています。

5案件継続制作
2022年〜継続中
EXPO連動展示会とバーチャルを接続

※件数は社内台帳（W009・W013・W014・W022・W028）に基づく

課題｜B2B製品・技術を、体験型で伝える場が必要

  • 展示会での製品訴求をバーチャルで補完・拡張したい
  • 幅広いプロダクトラインを、インタラクティブに体験できる形で提示したい
  • 地域連携・CSR活動も含めたブランドイメージの発信

往来のソリューション｜展示会連動から地域連携まで、多層的なワールド展開

  • JIMTOF2022に合わせてB2Bショールーム「Kyocera Tool World」を制作・展示会と連動
  • ジャパンドローン2023に合わせてレーザーコンセプト製品展示「Kyocera Laser World」を制作
  • 高耐久スマートフォン・物流体験を組み込んだ「Kyocera Mobile World」を制作（「スマート物流EXPO」東京ビッグサイト・2024.01.24-26と連動して公開）
  • 製造工程をインタラクティブに体験できる「ファインセラミックスワールド」を制作
  • 地域連携として京都府京田辺市・甘南備山をモチーフにした「ワールドポータル by Kyocera」を制作

往来の役割は、企画立案・クリエイターチームの組成・制作ディレクション。各ワールドの制作には案件ごとに最適なクリエイターを編成し、往来がプロデュースとして全体を統括する体制で、4年にわたる継続展開を支えています。

プロジェクトの歩み

  • 2022.11 「Kyocera Tool World」公開 — JIMTOF2022と連動したB2Bショールーム
  • 2023.06 「Kyocera Laser World」公開 — レーザー技術の体験型コンテンツ
  • 2024.01 「Kyocera Mobile World」公開 — 「スマート物流EXPO」（東京ビッグサイト・2024.01.24-26）のリアル展示と連動
  • 2024.08 「Kyocera Fine Ceramics World」「Kyocera World Portal」同時公開 — ファインセラミックス体験と、京セラの森をモチーフにしたハブワールド

成果｜産業XR活用を5案件継続、展示会連動でリアルとも接続

  • B2B製品の体験型展示という産業XR活用を5案件にわたり継続
  • 展示会連動によりリアルイベントとバーチャルを接続した複合施策を実現
  • 製品展示から地域連携まで、多角的なブランドコミュニケーションを展開
