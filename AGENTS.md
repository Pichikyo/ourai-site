# AGENTS.md — ourai-site

往来（OURAI）のコーポレートサイト。XR・AI・リアル空間を横断する体験設計スタジオの公式サイト。
このファイルは Codex 等の AI コーディングエージェント向けの入口です。Claude 向けの CLAUDE.md と同内容をツール中立に記述しています。両者が食い違ったら、下記の「正本」が勝ちます。

## 正本（ルールの一次情報源）

- **design.md** — デザイン方針の正本。色・フォント・字間・余白・改行ルール・全確定値。HTML/CSSを1行でも触る前に必ず読む。ページごとに別テーマを作らない
- **CLAUDE.md の「最重要の運用ルール」セクション** — 運用ルールの正本（公開範囲・数字の出典・触ってはいけないもの）
- 掲載してよい案件・数字は**社内台帳が正本**（このリポジトリの外にある。場所はオーナーに確認）

## 技術スタック

- 静的 HTML + CSS + バニラ JS。フレームワーク・ビルドなし
- 共通スタイルは style.css に集約（デザイントークンは :root、oklch）
- デプロイ先: Cloudflare Pages（本番: https://ourai-site.pages.dev）
- llms.txt / works.md / company-record.html は AI クローラー向け。ナビからリンクしない独立運用

## 最重要の運用ルール

1. **このフォルダの全ファイルはデプロイで公開される。** 台帳・計画書・レビューメモ等の内部資料を絶対に置かない（過去に事故あり）。置いてよいのは公開物と design.md / CLAUDE.md / AGENTS.md のみ
2. **実績の数字を独自に書き換えない。** 数字には出典または「自社調べ（時点）」表記が必須
3. **サイトに載っていない案件名を新たに載せない。** 掲載可否は社内台帳で確認が取れてから
4. **変更は PR（プルリクエスト）で提出する。** main への直接 push・直接デプロイはしない。承認とデプロイはオーナー（東 智美）が行う

## デザイン上の絶対禁止（design.md より抜粋）

- 紫は絶対 NG／グラデーション NG／box-shadow NG／ホバーリフト（translate/scale）NG
- 色は :root の oklch トークンのみ。生の色コードを書かない
- 見出し・リードの泣き別れ禁止（`.wb` 文節区切りを崩さない）
- 手動改行は `br-pc` クラス（PC専用）。素の `<br>` を本文に足さない
- type scale を勝手に下げない（可読性優先の営業判断で確定済み）

詳細と数値はすべて design.md 本文を参照すること。この要約だけで実装しない。

## 構造上、触ってはいけないもの（オーナーの明示指示がない限り変更禁止）

- `.env`（存在する場合、中身を開かない・変更しない）
- KV スライダー（index.html の `#kvTrack` 一式）の構造
- FEATURED 実績の構造（index.html の worksData 描画ロジック／works.html の特集カード群）。カードの**中身**の更新・追加は可、**構造**の変更は不可
- ハンバーガーメニュー（`#menuBtn` / `#mobileNav`）の構造

## 実行・確認

- ローカル確認: `python3 -m http.server 8742` をリポジトリ直下で
- コミット規約: Conventional Commits（feat:/fix:/refactor: 等）
